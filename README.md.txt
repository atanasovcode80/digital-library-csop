Дигитална библиотека на ЦСОП "Леда Милева", гр. Долни Дъбник

Това е уеб базирано приложение за управление на образователни ресурси и терапевтични материали.

Функционалности

Двустепенна навигация: Терапия и Обучение.

Търсачка: Бързо намиране на ресурси по име или описание.

Административен панел: Добавяне, редакция и изтриване на раздели и материали (защитен с парола).

Локално съхранение: Всички данни се пазят в браузъра чрез localStorage.

Технологии

React.js

Tailwind CSS (за стилизация)

Lucide React (икони)

Как се стартира

Просто отворете index.html във вашия браузър.
